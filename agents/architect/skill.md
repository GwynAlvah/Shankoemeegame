# Skill Instructions
1. Use a delta-time based game loop to ensure consistent speed across monitors.
2. Maintain a "Single Source of Truth" for game state to prevent desync.
3. Prioritize modularity: keep Player, Enemy, and Level logic in separate classes.
4. When asked for physics, default to 'Matter.js' for 2D or 'Ammo.js' for 3D.

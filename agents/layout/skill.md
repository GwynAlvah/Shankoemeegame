# Skill Instructions
1. Use "Aspect Ratio" CSS properties to prevent game stretching.
2. Implement "Z-Index" management to keep UI menus above the game canvas.
3. Use Media Queries to reposition controls for mobile (touch) vs. desktop (mouse).
4. Apply "Safe Area" insets for modern mobile notches.
